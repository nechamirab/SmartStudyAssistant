# Research And Legacy Code

This folder contains the older RAG experimentation platform that existed before the project was consolidated into the student-facing PDF study assistant MVP.

It includes legacy code for:

- LangChain RAG experiments
- benchmark runners and reports
- embedding providers
- vector stores
- retrieval and reranking experiments
- old app prototypes
- legacy tests and experimentation docs

The main product now lives in `ui/`, `pages/`, `services/`, and `core/`. Treat this folder as reference material unless you are specifically working on research or benchmark tasks.
